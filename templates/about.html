System Architecture

This project uses an unsupervised abnormality detection approach based on a convolutional autoencoder. The model is trained to reconstruct brain MRI scans and then compares the reconstructed image with the original image during inference.

The preprocessing pipeline includes image loading, skull stripping, intensity normalization, and resizing to the model input size. After reconstruction, the system generates a reconstruction error map, anomaly heatmap, heatmap overlay, and anomaly mask.

The final result is produced as Normal or Abnormal using reconstruction loss, anomaly score, and detected region size instead of supervised class prediction.

Implemented Features
Detection Workflow

Supports MRI image upload in PNG, JPG, JPEG, GIF, and BMP formats with autoencoder-based Normal or Abnormal detection.

Visual Evidence

Generates anomaly heatmap, heatmap overlay, binary anomaly mask, anomaly score, and reconstruction loss for each analysis.

Reporting and History

Stores analysis history and provides downloadable PDF reports with patient details and visual outputs.

Security and Roles

Includes role-based access for admin, doctor, and patient users, along with audit logging and encrypted backup support.
